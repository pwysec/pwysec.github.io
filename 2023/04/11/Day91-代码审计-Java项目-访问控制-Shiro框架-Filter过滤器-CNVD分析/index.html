-Filter过滤器

1、com.jsh.erp.filter.LogCostFilter

使用了4个if判断，

第1个if：如果有seesion则不跳登录页

第2个if：如果包含login或register页面则不跳登录页

第3个if：调用verify传入ignoredList与用户请求URL

第4个if：如果存在allowUrls则不跳登录

查找ignoredList来源

2、allowUrls存在ignoredList
如果访问css、js这种文件就不需要登录
